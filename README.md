# Site Kemp

Одностраничный лендинг для генерации текстовых работ.

## Технологии

- Next.js 14
- TypeScript
- Tailwind CSS

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
npm start
```
